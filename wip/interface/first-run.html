Select your wallet

Paper Wallet

A paper wallet is a method of storing your Stellite coins 'offline'. Once you've generated a paper wallet you need to store the information securely to later import into the GUI wallet for your Stellite to be usable.

 

GUI Wallet

The GUI wallet is a graphical wallet you'll install on your computer. You can use the GUI wallet to send and receive Stellite coins to other people or exchanges. You can create a new one or import an existing one.

Select a mining pool

A mining pool is where many different miners work together to find a block and share the reward. If you were
to mine on your own (solo mining) it could take weeks before you find a block, if you find a block.

The community has set up a bunch of pools, some of the most popular ones are listed below. We recommend picking
a pool that has some miners and a healthy hash rate. Picking a busy pool results in smaller, more frequent payouts while a smaller
pool results in larger, less frequent payouts.
