Frequently Asked Questions

I have my own xmr-stak or xmrig configuration already

You can use your existing (compatible) configuration for xmrig and xmr-stak by replacing the config files in 

installation-directory/miner

For the mining stats to display correctly you need to enable the API for each miner by doing the following:

xmrig Ensure you have the following in you config.json file 
"api":{
  "port":16000,
  "access-token":null,
  "worker-id":null
}
xmr-stak Ensure you have the following in you config.txt file 
"httpd_port" : 16000,
I'm on macOS or Linux and would like to use xmr-stak, xmrig-nvidia or xmrig-amd

If you already use one of these miners you can simply follow the answer for `I have my own xmr-stak or xmrig configuration already`.

If you don't have a version ready, you can follow the build instructions for xmrig and xmr-stak on their GitHub pages

I keep getting the 'no compatible miner found' error

Your antivirus most likely removed the miner before you could exclude the folder from being scanned. If you followed the antivirus guides in the initial setup you can simply extract the miner again, replacing the current files, and run the GUI miner again.

If you haven't exluded the miner's folder from being scanned, do so now and run the GUI miner again.

I'd like to use a pool that is not listed

We currently only list pools we have tested in the miner. You are free to change the config for xmrig and xmr-stak to rather use a pool you want. You can track the progress on this feature here

xmrig Edit the config.json file in the miner folder 
"pools":[
	{
		"url":"pool.example:5555",
		"user":"your-stellite-address",
		"pass":"Stellite GUI Miner",
		"keepalive":true,
		"nicehash":false,
		"variant":1
	}
],
xmr-stak Edit the pools.txt file in the miner folder 
"pool_list":
[
  {
    "pool_address" : "pool.example:3333",
    "wallet_address" : "your-stellite-address",
    "rig_id" : "",
    "pool_password" : "Stellite GUI Miner",
    "use_nicehash" : false,
    "use_tls" : false,
    "tls_fingerprint" : "",
    "pool_weight" : 1
  },
],
"currency" : "stellite",
